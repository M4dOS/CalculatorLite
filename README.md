# CalculatorLite
## Cделана данная штука для одной из моих лабораторных работ
Здесь реализованы основные штуки обычного калькулятора и даже слегка больше
## Как пользоваться?
Есть циферблат, есть операции, есть даже одноместный буфер, если нужно запомнить какое-то значение и всё вместе посчитать
## Функции
- Сложение, вычитание, умножение, деление
- Реверс числа, математический корень и квадрат
- Возведение в корень и степень произвольного числа
- Работа с памятью (пока одноместный): прибавление
- Копирование из поля и из памяти, вставка в поле
